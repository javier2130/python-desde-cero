# 🐍 Python desde cero

Este repositorio contiene el código de la serie de YouTube donde aprendemos programación en Python desde cero.

---

## 📺 Video 01: Tu primer programa en Python

[![Ver en YouTube](https://img.shields.io/badge/YouTube-Ver%20Video-red?logo=youtube&logoColor=white)](LINK_DEL_VIDEO)

En este video aprenderás:

- Mostrar mensajes con `print`
- Pedir datos con `input`
- Usar variables
- Trabajar con tipos de datos (`int`, `string`)
- Tomar decisiones con `if / else`
- Repetir acciones con `for`

📁 Carpeta del video:
video_01_primer_programa/
